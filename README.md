# 📚 Book Recommendation System (ML Project)

A Machine Learning-based Book Recommendation System built using Collaborative Filtering and Cosine Similarity.

## Features

- Personalized book recommendations
- Book cover previews
- Interactive Streamlit web interface
- Similarity-based recommendation engine
- Clean and responsive UI

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## How It Works

1. Data preprocessing and cleaning
2. User-book matrix creation
3. Collaborative filtering
4. Cosine similarity calculation
5. Recommendation generation

