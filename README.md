# 📚 Book Recommendation System (ML Project)

A Machine Learning based Book Recommendation System built using Collaborative Filtering and Cosine Similarity.

## Features
- Personalized book recommendations
- Book cover previews
- Streamlit web app
- Similarity based recommendations

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
